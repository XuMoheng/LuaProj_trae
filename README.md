# Lua Compiler (C++)

A basic Lua compiler implementation in C++ from scratch generated by trae.

## Features
- **Lexer**: Tokenizes Lua source code (keywords, identifiers, numbers, strings, operators).
- **Parser**: Recursive descent parser that generates an Abstract Syntax Tree (AST).
- **AST Printer**: Visualizes the parsed AST.

## Requirements
- C++17 or later
- CMake 3.10+

## Build
```bash
mkdir build
cd build
cmake ..
make
```

## Usage
Run the compiler with a Lua script:
```bash
./lua_compiler path/to/script.lua
```
Or run in REPL mode (basic lexing/parsing verification):
```bash
./lua_compiler
```
